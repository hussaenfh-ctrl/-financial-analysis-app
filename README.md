# برنامج التحليل المالي

أداة تحليل مالي تعمل بالكامل داخل المتصفح (بدون خادم/بدون رفع بياناتك لأي مكان) لعرض:

- **النسب المالية**: السيولة، النشاط والكفاءة، الربحية، المديونية
- **التحليل الرأسي** (Vertical Analysis)
- **التحليل الأفقي** (Horizontal Analysis)
- **دورة التحويل النقدي CCC** (Cash Conversion Cycle)
- **خلاصة الوضع المالي والرأي الفني** مع نقاط القوة والضعف (مولّدة آليًا بقواعد منطقية)

يمكن إدخال البيانات **يدويًا** عبر جدول تفاعلي، أو **برفع ملف Excel/CSV بأي شكل** ثم مطابقة صفوف الملف وأعمدته يدويًا مع بنود القوائم المالية.

> ⚠️ تنبيه: هذا تحليل آلي إرشادي مبني على البيانات المُدخلة فقط، وليس استشارة مالية أو استثمارية معتمدة.

## الاستخدام

1. افتح الرابط المنشور (أو ملف `index.html` مباشرة في المتصفح).
2. من تبويب **"إدخال البيانات"**: اختر إدخال يدوي أو رفع ملف إكسيل.
   - **إدخال يدوي**: اضغط "+ إضافة فترة / سنة" ثم أدخل أرقام قائمة الدخل والمركز المالي. البنود المحسوبة (الإجماليات) تُحدَّث تلقائيًا.
   - **رفع إكسيل**: ارفع ملفك، اختر الورقة، حدد أعمدة الفترات، ثم طابق كل بند بصفه المناسب في الملف، واضغط "تطبيق المطابقة".
3. تصفح باقي التبويبات: النسب المالية، التحليل الرأسي، التحليل الأفقي، دورة CCC، والخلاصة.
4. زر "تحميل بيانات تجريبية" في الأعلى يملأ التطبيق ببيانات افتراضية لثلاث سنوات لتجربة كل المميزات فورًا.
5. زر "طباعة / PDF" لتصدير النتائج كتقرير.

بياناتك تُحفظ محليًا في متصفحك (`localStorage`) ولا تُرسل لأي خادم خارجي.

## النشر على GitHub Pages

المشروع **static بالكامل** (HTML/CSS/JavaScript فقط، بدون أي خطوة بناء/Build)، لذلك ينشر مباشرة على GitHub Pages:

```bash
git init
git add .
git commit -m "النسخة الأولى من برنامج التحليل المالي"
git branch -M main
git remote add origin https://github.com/USERNAME/REPO_NAME.git
git push -u origin main
```

بعدها من صفحة الـ repo على GitHub:

1. **Settings → Pages**
2. تحت "Build and deployment" اختر **Source: Deploy from a branch**
3. اختر **Branch: main** و **Folder: / (root)**
4. اضغط **Save** وانتظر دقيقة أو اثنين

هيظهر رابط الموقع في نفس الصفحة بالشكل:
`https://USERNAME.github.io/REPO_NAME/`

## هيكل المشروع

```
index.html              الصفحة الرئيسية
css/style.css           التنسيق
js/dataModel.js         بنود القوائم المالية + إدارة الفترات + التخزين المحلي
js/analysis.js          حساب النسب + التحليل الرأسي/الأفقي + CCC
js/narrative.js         مولّد الخلاصة والرأي الفني
js/excelImport.js       قراءة ملفات إكسيل ومطابقتها يدويًا
js/charts.js            الرسوم البيانية
js/app.js               ربط الواجهة وإدارة الحالة
sample-data.js          بيانات تجريبية
```

المكتبات الخارجية (منسوخة محليًا في `vendor/`، لا حاجة لـ Node.js أو npm ولا اتصال إنترنت وقت الاستخدام):
[SheetJS](https://sheetjs.com/) لقراءة ملفات إكسيل، و[Chart.js](https://www.chartjs.org/) للرسوم البيانية.
